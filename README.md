# Climate & Disease Intelligence Dashboard

A machine learning-based system to analyze the impact of climate variability on disease patterns like Dengue, Malaria, and Cholera.

## Features
- Climate trend analysis (Temperature, Rainfall, Humidity)
- Disease trend visualization
- Correlation & lag analysis
- PCA dimensionality reduction
- ML Models (Random Forest, XGBoost, Linear Regression)
- ARIMA forecasting
- Scenario-based predictions

## Tech Stack
- HTML, CSS, JavaScript
- Chart.js
- Machine Learning Models

## Results
- Random Forest achieved R² = 0.992
- Strong seasonal dependency observed
- Rainfall impacts Dengue with 1-month lag
